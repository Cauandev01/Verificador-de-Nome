# Verificador de Nome

Este é um projeto simples desenvolvido com **HTML, CSS e JavaScript** que analisa um nome digitado pelo usuário e retorna algumas informações sobre ele.

## 🚀 Funcionalidades

Ao digitar um nome e clicar no botão **Verificar**, o site mostra:

* O nome digitado
* Quantidade total de caracteres
* A segunda letra do nome
* Primeiro índice da letra **"a"**
* Último índice da letra **"a"**
* As últimas 3 letras do nome
* Separação das palavras do nome
* Nome em **letras maiúsculas**
* Nome em **letras minúsculas**

## 🛠 Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## 📚 Conceitos de JavaScript utilizados

Neste projeto foram praticados alguns conceitos importantes da linguagem:

* Manipulação do **DOM**
* Eventos com `addEventListener`
* Template Strings
* Métodos de string como:

  * `length`
  * `indexOf()`
  * `lastIndexOf()`
  * `slice()`
  * `split()`
  * `toUpperCase()`
  * `toLowerCase()`

## 🎯 Objetivo do projeto

Este projeto foi criado com o objetivo de **praticar JavaScript básico**, especialmente manipulação de strings e interação com elementos da página.
